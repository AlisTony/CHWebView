# CHWebView
封装iOS UIWebView常用功能的基础控件

# 安装
利用pod安装或者下载工程文件把CHWebView的文件夹拖入工程中
```
pod 'CHWebView'

```
# 实现功能
1.加载web页面时显示进度条
2.初始化方法拓展增加加载本地文件和远端文件两种方式
3.JS调用OC方法封装.
